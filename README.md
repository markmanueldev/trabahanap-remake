# TrabaHanap Remake

## Get started


1. Install Docker and pnpm package manager
2. Run Docker Compose

   ```bash
   docker compose up -d
   ```
3. Go to client and install dependencies

   ```bash
   pnpm install
   ```
4. Likewise go to server and install dependencies

   ```bash
   pnpm install
   ```

## To test all files 

1. Go to server dir
2. Test files in this format

   ```bash
   pnpm test 
   ```
